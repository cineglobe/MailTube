# Contributing

Open an issue before a large change. Keep pull requests focused, add tests for behavior changes, and never commit personal configuration, cookies, email addresses, downloaded media, or `my_version/`.

Development requires Python 3.12+, Node 22+, pnpm, ffmpeg, and Deno. Follow [docs/development.md](docs/development.md), run the complete local checks, and include screenshots for dashboard changes. Contributions are licensed under AGPL-3.0-only.

Be specific in bug reports: platform, architecture, MailTube version, sanitized logs, and reproduction steps. Never post credentials, signed URLs, cookies, private video URLs, or sender addresses.
